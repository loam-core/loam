# Code of Conduct

Loam Core follows the spirit of the Contributor Covenant, adapted for a small,
early-stage open-source project.

## Our Pledge

We aim to make participation in Loam Core respectful and harassment-free.

## Standards

Positive behavior includes:

- respectful communication
- constructive feedback
- helping newcomers

Unacceptable behavior includes:

- harassment
- personal attacks
- trolling
- publishing private information

## Enforcement

For concerns or violations, please report the issue privately using GitHub’s reporting tools:

https://github.com/loam-core/loam/issues/new/choose

Do not email the maintainers directly.
Do not open public issues for sensitive reports.

Reports will be handled confidentially.
